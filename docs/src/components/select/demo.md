## 选择器尺寸

通过 `size` 属性设置输入框的尺寸，支持 4 种尺寸，分别是 `mini` - <b>迷你</b>，`small` - <b>小型</b>，`default` - <b>普通（默认）</b>，`large` - <b>大型</b>。
